# Software Testing & API Automation Practice

This repository contains my practical work created to improve my skills in software test engineering.
The work includes applications on basic testing principles, the HTTP protocol, REST API testing and test automation with Python.

## Scope

The scopeof this project is:
- To learn software testing processes through practical application
- To automate manual testing logic
- To test API services
- To develop a Python-based test automation infrastructure
- To understand a testing approach that can be used in real projects

  
## Technologies Used

- Python 3.x
- PyTest
- Requests Library
- JSON
- REST API
- HTTP Protocol
- Git / GitHub
